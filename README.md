# Nexplore-Test

I create 3 repositories:

- Http Backend App
- GraphQL Backend App
- React Frontend (works with Http Backend App)

## Create Database Table

```bash
CREATE TABLE duties(
    id PRIMARY KEY,
    name VARCHAR(40)
)
```

## Install Apps

### HTTP BACKEND APP

Folder: nexplore-backend

1. Install modules

```bash
npm i
```

2. Set .env variables with your host and db params

3. Start app

```bash
npm run start
```

### GRAPHQL BACKEND APP
